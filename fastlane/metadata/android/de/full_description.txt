Diese App zeigt die aktuellen Corona-Daten, z.B. Gesamtinfektionen, Todesfälle, etc., der jeweiligen Gebieten in Deutschland an.

Zusätzlich werden die Farben der Corona-Ampel angezeigt um auf einen Blick Klarheit über mögliche Einschränkungen zu haben(7-Tage-Inzidenz pro 100.000 EW <35: Grün ; <50: Gelb ; >50: Rot ; In Bayern >100: Dunkelrot).

Es werden immer nur die aktuellen Daten angezeigt, keine Verläufe über mehrere Tage.

<strong>Als Datenquellen werden verwendet:</strong>

* Für <a href="https://www.lgl.bayern.de/gesundheit/infektionsschutz/infektionskrankheiten_a_z/coronavirus/karte_coronavirus/index.htm">bayrische Städte und Landkreise</a> die Daten des <a href="https://www.lgl.bayern.de/">LGL Bayern</a>
* Für alle anderen <a href="https://npgeo-corona-npgeo-de.hub.arcgis.com/datasets/917fc37a709542548cc3be077a786c17_0">Städte, Landkreise</a>, <a href="https://npgeo-corona-npgeo-de.hub.arcgis.com/datasets/ef4b445a53c1406892257fe63129a8ea_0">Bundesländer</a> sowie für das <a href="https://www.rki.de/DE/Content/InfAZ/N/Neuartiges_Coronavirus/Fallzahlen.html">gesamtdeutsche Infektionsgeschehen</a> die Daten des <a href="https://www.rki.de/">Robert Koch Instituts</a>
* Für die weltweite Statistik die Daten von <a href="https://www.worldometers.info/">worldometers.info</a>

<strong>Bibliothek:</strong>

Das Auslesen der Daten wurde in eine separate Bibliothek ausgelagert, die außerdem noch die Daten der <a href="https://www.jhu.edu/">Johns Hopkins University</a> und weitere Daten von <a href="https://www.worldometers.info/">worldometers.info</a> auslesen kann.

Sie darf selbstverständlich auch in anderen Projekten verwendet werden.
